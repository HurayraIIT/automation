# automation
Automation Testing.


## Some useful commands

Install playwright:

```
npm init playwright@latest
```

To update playwright:

```
npm install -D @playwright/test@latest
```

Usually after Playwright update, browsers need to be updated with command:

```
npx playwright install
```
